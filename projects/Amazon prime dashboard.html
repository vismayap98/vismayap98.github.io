Amazon Prime Dashboard

  • Tech Stack: Power BI, DAX, Data Modeling, Data Visualization, Azure, SQL
  • Github URL: Project Link

Designed an interactive Power BI dashboard for Amazon Prime, focused on real-time analysis of user engagement, content trends, and subscription metrics. The dashboard facilitates personalized insights that empower stakeholders to optimize their content strategy effectively.

Key features of the project include:

  • Interactive filters for dynamic data visualization, allowing users to segment data by demographics, viewing habits, and subscription status.
  • Real-time analytics to track user engagement metrics, including watch time, content popularity, and churn rates.
  • Trend analysis of content performance over time to identify seasonal patterns and emerging viewer preferences.
  • Visualizations using bar charts, line graphs, and maps to present data in an easily interpretable format.
  • Integration with Azure for seamless data refresh and up-to-date reporting capabilities.

The dashboard enables stakeholders to make data-driven decisions regarding content creation and marketing strategies, ultimately enhancing user satisfaction and retention.
