Visitor Insights

Track real-time user behavior, engagement, and interaction across my personal portfolio.

This project showcases how I implemented a live visitor tracking solution using Google Analytics 4 and Looker Studio. Custom events were sent using JavaScript's IntersectionObserver to track scroll-based section views. The data was then visualized dynamically to understand which parts of my portfolio attract the most attention.
