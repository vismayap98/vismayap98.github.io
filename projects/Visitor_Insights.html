Visitor Insights

  • Tech Stack: Google Analytics 4, Looker Studio, JavaScript, HTML
  • Link: View in Portfolio

This project showcases a real-time visitor tracking solution integrated directly into my personal portfolio website. I used Google Analytics 4 to track behavior and custom section views via JavaScript's IntersectionObserver, and visualized the data using Looker Studio.

  • Implemented custom section_view events to track when users scroll into key sections of the site.
  • Created real-time dashboards showing device category, session trends, country of origin, and engagement.
  • Configured GA4 custom dimensions to capture section names and enabled rich reporting.
  • Added a scroll-based observer to dynamically push view data into GA4 with zero latency.
  • Embedded the Looker Studio dashboard seamlessly into the portfolio for live reporting.

How to View

You can view the dashboard under the "Visitor Insights" section of my portfolio website or access it directly via the link above.

Key Insights

  • Geographic and device-based visitor distribution across portfolio sections
  • Real-time session activity, engagement rate, and returning users
  • Most viewed content blocks based on custom section interactions
