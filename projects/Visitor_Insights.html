Visitor Insights

  • Tech Stack: Google Analytics 4, Looker Studio, JavaScript, HTML, GA4 Events
  • Link: Project Section

This project showcases a real-time visitor tracking solution implemented on my personal portfolio site. Custom section view tracking was implemented using Google Analytics 4 and JavaScript's IntersectionObserver. The data is collected and visualized using Looker Studio for dynamic analysis of audience behavior.

Key features of the project include:

  • Custom section-based view tracking using IntersectionObserver and GA4 event tags.
  • Real-time reporting of sessions, locations, devices, and engagement using Looker Studio.
  • GA4 filters to exclude bots and unengaged users for clean analytics.
  • Visual insights on most visited sections, country-level traffic, and device usage.
  • Accessible directly via the "Visitor Insights" section of my personal portfolio website.

This solution helps understand user attention patterns across different content sections without using cloud-based resources like AWS or serverless Lambda logic.
