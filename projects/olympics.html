Olympics 2024 Dashboard

  • Tech Stack: Power BI, DAX, Data Modeling, Data Visualization, Azure, SQL
  • Project Link: Project Link

Developed an interactive Power BI dashboard for the Olympics 2024, providing real-time insights into medal counts, athlete performance, and country rankings. The dashboard is designed to facilitate data-driven decisions for enhancing Olympic strategies and performance analysis.

Key features of the project include:

  • Visualization of the sum of medals won by sport, enabling a comparative analysis of different disciplines.
  • Interactive charts showing the percentage of medals by country, helping stakeholders identify leading nations in various sports.
  • Detailed statistics on the average number of medals earned by gender, highlighting participation equity in the games.
  • A geographic map overview displaying medal distributions globally, providing insights into regional performance.
  • Comprehensive breakdown of medals by type (bronze, silver, gold), aiding in the analysis of overall performance trends.

Key Insights

  • Sum of Medals by Sports: Detailed analysis of medals won across various disciplines.
  • Average of Medals Earned by Gender: Insights into gender participation in medal winning.
  • Map Overview: Visualization of countries' performances on a global map.
  • Total Medals: Breakdown of total medals including bronze, gold, and silver for each participating country.

This dashboard provides a comprehensive view of the Olympics 2024, allowing users to track performance metrics and trends, ultimately supporting the optimization of training and competitive strategies.
