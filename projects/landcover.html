Landcover Classification of Satellite Imagery Using Deep Learning

Developed a 2D-3D hybridization CNN model using Python (NumPy, Pandas, Seaborn) for satellite image classification, achieving high accuracy in vegetation classification using GPU acceleration.

This web-based application leverages advanced deep learning techniques to analyze and classify satellite imagery effectively. The model's architecture combines 2D and 3D convolutional layers, allowing it to learn spatial features in satellite data.

In addition, the scalability of the web application was tested by generating 10,000 user requests with the help of Apache Beam, ensuring robust performance and reliability under high load conditions.

Tech Stack:

  • Python (NumPy, Pandas, Seaborn)
  • Deep Learning (CNNs)
  • GPU Acceleration
  • Apache Beam for Load Testing
